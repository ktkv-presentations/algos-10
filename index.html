JSON

JavaScript Object Notation

JSON - индустриальный стандарт для хранения данных
JSON в ЯП представляется как строка

структуры данных, заключенные в фигурные скобки {}, где каждый элемент представлен в формате "ключ: значение"


{
    "name": "John",
    "age": 25,
    "married": false,
    "children": ["Anna", "Tom"],
    "address": {
        "city": "New York",
        "zip_code": "10001"
    }
}

JSON используется

Веб-разработка

JSON часто используется для передачи данных между сервером и клиентом, особенно в AJAX-запросах и API

Конфигураций

JSON используется для хранения настроек в файлах конфигурации в различных приложениях

Передача данных в API

JSON является стандартным форматом для большинства веб-сервисов и API

JSON в PYTHON

Для работы с json необходимо использовать встроенную библиотеку json

import json

json.dumps(data) превращает словарь в строку json

json.loads(json) превращает строку json в словарь


import json

# Преобразование Python словаря в JSON строку
data = {
    "name": "John",
    "age": 30,
    "city": "New York"
}
json_string = json.dumps(data)
print(json_string)

# Преобразование JSON строки обратно в Python словарь
data_from_json = json.loads(json_string)
print(data_from_json)

Методы словарей

Безопасное получение значения

my_dict.get(название_ключа, значение_если_отсутствует) 

my_dict = {"name": "Alice", "age": 30}
print(my_dict["city"]) # KeyError
print(my_dict.get("name"))  # Вывод: Alice
print(my_dict.get("city", "Unknown"))  # Вывод: Unknown

Методы словарей

Добавление в словарь новой пары "ключ-значение"

my_dict.update(словарь) 

my_dict = {"name": "Alice"}
my_dict.update({"city": "New York", "age": 31})
print(my_dict) # {"name": "Alice", "city": "New York", "age": 31}

Методы словарей

Удалить ключ и вернуть значение

my_dict.pop(название_ключа, значение_если_отсутствует) 

my_dict = {"name": "Alice", "age": 30}
age = my_dict.pop("age", "Not Found")
print(age)  # Вывод: 30
print(my_dict)  # Вывод: {'name': 'Alice'}

Методы словарей

Удалить последний добавленный ключ и вернуть значение

my_dict.popitem() 

my_dict = {"name": "Alice", "age": 30}
my_dict["city"] = "New York"
my_dict.popitem() # ("city": "New York")

Методы словарей
